# 🚀 PMO Status Automator (V1.0)
*Automating Project Governance with Python*

### 📋 Business Problem
In high-volume Project Management Offices, status reporting often consumes 5-10 hours of manual work per week. This script automates the formatting of risk logs and budget snapshots.

### 🛠️ Technical Stack
- **Language:** Python 3.10
- **Concepts:** Lists, f-strings, Variable Management, Boolean Logic.

### 📈 Business Impact
- **Time Saved:** Estimated 75% reduction in manual report formatting.
- **Accuracy:** Eliminates human error in budget-to-actual variance reporting.

### 🚀 How to Run
1. Open in Python environment.
2. Update the `project_list` variables.
3. Run the script to generate an 'Executive Summary'.
